## Descargador de vídeos de youtube


1. Clona el repo con
```
git clone https://github.com/voltios/youtubedownloader
```
2. Ejecuta este comando:
```
  python -m venv venv
```
3. Ejecuta este comando:
```
  ./venv/scripts/activate
```
4. Ejecuta:
```
  pip install yt-dlp
```
5. Y por último:
```
   python.exe main.py
   ```
